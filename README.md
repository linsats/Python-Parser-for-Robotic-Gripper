# Python URDF Parser For Robotic Gripper

## Get Started
1.Initialize repository
```
git clone https://github.com/stanford-iprl-lab/UniGrasp.git
```

2.Install requirement
```
pip3 install -r requirement.txt
```

3.Run
```
python gripper_files.py
```
